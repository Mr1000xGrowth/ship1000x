# Ship1000x

> Track what you actually ship with AI.

Local-first tracker for AI-assisted development. Measures focus time, code produced (real vs generated), and LLM cost across Claude Code, Codex, Cursor, git, and shell activity.

## Status

`v0.1.0` — early alpha, under active development. API may change.

## Install

```bash
pip install ship1000x
ship1000x init
ship1000x dashboard
```

Full documentation coming with `v0.2.0`.
